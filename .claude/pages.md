# 페이지 명세

## 인증

### `/login`
- **그룹**: `(auth)` - 하단 네비 없음
- **기능**: 로그인 폼
- **리다이렉트**: 로그인 성공 시 `/predict`로 이동

---

## 메인 탭 (하단 네비 포함)

### `/predict` - 예측 탭

**기능**:
- 전략별 예측 종목 리스트 표시
- 각 전략 섹션으로 구분
- 종목 클릭 시 `/predict/[stockId]`로 이동

**데이터**:
- 전략 목록
- 전략별 예측 종목 (종목코드, 종목명, 예측가, 현재가 등)

**UI 구성**:
```
[전략 A]
├── 종목1 카드
├── 종목2 카드
└── 종목3 카드

[전략 B]
├── 종목4 카드
└── 종목5 카드
```

---

### `/predict/[stockId]` - 종목 상세

**기능**:
- 종목 상세 정보 표시
- 차트 구현 (캔들차트 또는 라인차트)
- 호가창 구현

**데이터**:
- 종목 기본 정보 (코드, 이름, 현재가, 등락률)
- 차트 데이터 (OHLCV)
- 호가 데이터 (매수/매도 호가, 잔량)
- AI 예측가

**UI 구성**:
```
[헤더: 종목명, 현재가, 등락률]
[차트 영역]
[호가창]
[AI 예측 정보]
```

---

### `/balance` - 잔고 탭

**기능**:
- 당일 총 매입/매도 금액
- 현재 보유 잔고
- 수익률 분석

**데이터**:
- 당일 매입 총액
- 당일 매도 총액
- 보유 종목 리스트 (종목, 수량, 평균단가, 현재가, 수익률)
- 총 수익률

**UI 구성**:
```
[요약 카드: 총 매입 | 총 매도 | 수익률]
[보유 종목 리스트]
├── 종목1 (수량, 수익률)
├── 종목2 (수량, 수익률)
└── ...
```

---

### `/history` - 내역 탭

**기능**:
- 달력으로 날짜 선택
- 잔고 누적 그래프
- 보유 잔고 출력 (최신화)
- 날짜 클릭 시 `/history/[date]`로 이동

**데이터**:
- 달력 데이터 (거래 있는 날 표시)
- 잔고 추이 데이터 (일별 총 자산)
- 현재 보유 잔고

**UI 구성**:
```
[달력 컴포넌트]
[잔고 누적 그래프]
[현재 보유 잔고 요약]
```

---

### `/history/[date]` - 일별 상세

**기능**:
- 선택한 날짜의 매매 내역
- 종목별 통계

**데이터**:
- 해당일 매매 내역 (종목, 매수/매도, 수량, 가격, 시간)
- 종목별 수익률

**UI 구성**:
```
[날짜 헤더]
[당일 요약: 매수 N건, 매도 N건, 수익률]
[종목별 통계 리스트]
├── 종목1: +5.2% (매수 2건, 매도 1건)
├── 종목2: -1.3% (매수 1건)
└── ...
[상세 거래 내역]
```

---

### `/settings` - 설정 탭

**기능**:
- 계좌 연동 설정
- 로그인 정보 관리
- 매매 설정값 (유저별)
- 로그아웃

**데이터**:
- 연동된 계좌 정보
- 사용자 정보
- 매매 설정값 (손절/익절 비율 등)

**UI 구성**:
```
[계좌 연동 섹션]
[매매 설정 섹션]
├── 손절 비율
├── 익절 비율
└── 기타 설정
[계정 섹션]
├── 로그인 정보
└── 로그아웃 버튼
```
